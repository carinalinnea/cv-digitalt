UX/UI - designer
Carina Appelgren

Jag planerar och organiserar tjänster som skapar goda användarupplevelser på webben. Jag kartlägger hur användarnas behov kopplas ihop med verksamhetsmålen och lägger sedan fram förslag som omvandlas till användbara gränssnitt, med stort fokus på tillgänglighet.

Design & content

Min önska är att alltid jobba med egna bilder och att ta fram förslag som uppfyller de önskemål som finns från kundens håll. Vi gör en plan och går tillsammans framåt för att nå dit, alltid lyhörd och anpassningsbar. Jag hjälper till att ta fram innehåll och text till uppdraget och gör de undersökningar som behövs för att få en bra start och grund att jobba vidare från.

UI - design

Här fokuserar jag på produkter och systems utseende och känsla, hur skärmen ser ut och fungerar. UI syftar på att enbart visuellt vägleda användaren genom en produkt eller systems användargränssnitt. Ett UI element kan vara en knapp, en skärm, ett ljud eller en sida. Det är något användaren kan trycka på, se, höra eller interagera med för att kunna göra ett informationsutbyte.

UX - design

Mitt arbete som UX-designer är att utveckla produkter som upplevs som maximalt användarvänliga, responsiva, smarta, smidiga och snabba. Som UX-designer arbetar jag med tester av wireframes, mockups och prototyper.
